What is  FDA-GO ?



FDA-GO provides users with drug search capabilities to view adverse events reported by patients, drug labeling information and reported drug, medical device, and food recalls.

Getting Started

To begin your drug search, simply enter drug search criteria within the search field on the main page and click Go. Drug search is also accessible on the results page. NOTE: All results will only return a maximum of 100 items.

To quickly retrieve the latest drug, medical device, and or food recalls, select one of the recall links located on the main page or on the results page.

Disclaimer

This is a prototype application designed to take on the challenge from the FDA to the developer community to take advantage of ther new open API for public data to improve public health. This application is solely for the purpose as a proof-of-concept and uses the beta API provided by OpenFDA. All data is provided by the API and complies with the FDA's terms of service.

See open.fda.gov for more information.
